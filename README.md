# GTMサービス ランディングページ

Googleタグマネージャー（GTM）の新規作成と修正をサービスとして提供するプロフェッショナルなランディングページです。

## 📋 プロジェクト概要

このプロジェクトは、GTMの導入・運用支援サービスを提供する企業向けのランディングページです。訪問者に対してサービス内容を明確に伝え、問い合わせを促進することを目的としています。

## ✨ 実装済み機能

### 1. レスポンシブデザイン
- デスクトップ、タブレット、モバイルすべてのデバイスに対応
- ハンバーガーメニューによるモバイルナビゲーション
- 768px以下でモバイルレイアウトに切り替わり

### 2. ヒーローセクション
- 魅力的なグラデーション背景
- アニメーション付きアイコン
- CTAボタン（無料相談・サービス詳細）

### 3. サービス紹介
- 4つの主要サービスカードを表示
  - GTM新規作成
  - GTM修正・最適化
  - コンバージョン計測
  - GTMトレーニング
- ホバーアニメーション付き

### 4. 特徴セクション
- 6つの選ばれる理由を表示
- アイコンとテキストで視覚的に訴求

### 5. 料金プラン
- 3つの料金プラン（ベーシック、スタンダード、プレミアム）
- おすすめバッジ表示
- 各プランの機能リスト

### 6. お問い合わせフォーム
- 入力項目：
  - お名前（必須）
  - 会社名
  - メールアドレス（必須）
  - 電話番号
  - ご希望のサービス（必須）
  - お問い合わせ内容（必須）
- フォームバリデーション
- 送信時の通知表示
- リアルタイム入力フィードバック

### 7. インタラクティブ機能
- スムーズスクロール
- スクロールアニメーション
- フォーム送信後の成功通知
- ヘッダーのスクロール効果

### 8. フッター
- サービスリンク
- 会社情報リンク
- コピーライト表示

## 🚀 機能詳細

### JavaScriptの主な機能

1. **モバイルメニュー**: トグル機能とクリック時の自動クローズ
2. **スムーズスクロール**: アンカーリンクのスムーズな遷移
3. **フォーム処理**: バリデーション、送信処理、通知表示
4. **スクロールアニメーション**: Intersection Observer APIを使用した要素の表示アニメーション
5. **ヘッダー効果**: スクロール位置に応じたシャドウ変更

## 🔧 カスタマイズ方法

### 料金プランの変更
`index.html` の料金セクション（`id="pricing"`）で各プランの内容と価格を編集できます。

### サービス内容の追加
`index.html` のサービスグリッド（`.service-grid`）に新しい `.service-card` を追加できます。

### カラースキームの変更
`css/style.css` の `:root` セクションでカラー変数を変更できます：

```css
:root {
    --primary-color: #4285f4;
    --secondary-color: #34a853;
    /* その他のカラー変数 */
}
```
